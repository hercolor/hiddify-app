# 蝴蝶加速客户端二开总览

## 目标

蝴蝶加速客户端是面向自有 XBoard 后端的商业客户端。二开目标不是保留原通用配置客户端能力，而是把用户路径收敛为：登录账号 → 同步节点 → 查看会员 → 选择节点 → 一键加速。

## 用户侧页面

### Home

- 显示连接状态。
- 显示当前节点。
- 提供一键加速/断开。
- 非会员、会员到期、流量耗尽时提示开通或续费，不触发真实连接。

### Nodes

- 展示订阅返回的节点列表。
- 支持节点切换。
- 支持节点延迟测试。
- 节点展示使用安全名称，不暴露服务器地址、端口、协议细节。

### Membership / 我的账号

- 登录页：邮箱或手机号 + 密码。
- 注册页：邮箱、密码、确认密码、邀请码、协议确认。
- 忘记密码：邮箱/手机号验证码重置。
- 会员卡片：显示套餐徽标、到期时间、设备数。
- 安全中心：修改密码、绑定/更换手机号。
- 退出登录后回到登录界面。

## 平台行为

### Android

- 包名：`pro.y88.accelerator`。
- 图标来自项目根目录 `icon.png` 派生资源。
- 启动页暂为白屏。
- TUN/VPN 权限请求由连接流程触发。

### Windows

- 可执行文件名：`蝴蝶加速.exe`。
- 支持托盘。
- 关闭按钮弹窗允许最小化到托盘或退出。
- 安装包使用 Inno Setup，便携包为 release 目录 zip。
- 普通界面隐藏技术配置项。

### iOS / macOS

- iOS 主 App Bundle ID：`pro.y88.hudiejiasu`。
- iOS Network Extension：`pro.y88.hudiejiasu.PacketTunnel`。
- 需要 Apple Developer 账号完成正式签名分发。
- 当前启动页白屏。

## 安全与合规边界

- 不实现隐藏流量、反检测、绕过审核相关逻辑。
- 不在普通 UI 暴露订阅 URL、节点密码、协议、DNS、fake-ip、IPv6。
- 不记录用户浏览内容。
- 诊断日志和埋点必须脱敏。
- 生产密钥、用户 token、订阅 URL 不得硬编码进源码。

## 与上游的关系

本项目仍保留部分上游目录、类名、生成代码和 core binding 名称，例如 `hiddifycore`、`HiddifyCoreNativeLibrary`。这些属于内部实现命名，不代表当前产品对外仍使用原上游品牌。后续如需进一步清理，应按“可构建、可回滚、小步提交”的原则逐步重命名。
